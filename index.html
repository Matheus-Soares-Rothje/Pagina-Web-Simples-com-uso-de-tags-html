História da Web

(Voltar)

A evolução dessas tecnologias ocorreu de forma interligada:

  1. Necessidade de Comunicação:

    O Surgimento do TCP/IP (Anos 1970/80)O Problema:

    Na década de 1970, o Departamento de Defesa dos EUA criou a ARPANET, mas logo se deparou com um obstáculo.

    Redes diferentes (via rádio, satélite e cabo) não conseguiam conversar entre si.



  2. A Solução:

    Em 1974, os cientistas Vint Cerf e Bob Kahn publicaram um artigo que propunha o protocolo TCP/IP (Transmission Control Protocol / Internet Protocol).

    O IP funcionava como um "endereço" para identificar cada computador, e o TCP empacotava as informações para que nada se perdesse na rota.

    Em 1º de janeiro de 1983, a ARPANET adotou oficialmente esse protocolo, marcando o nascimento da internet moderna.



  3. A Criação da World Wide Web (1989)

    O Contexto:

    O TCP/IP conectava os computadores, mas acessar dados exigia conhecimentos técnicos complexos (como comandos para transferir arquivos via FTP).

    A Solução: Em 1989, o cientista britânico Tim Berners-Lee, trabalhando no laboratório europeu CERN, inventou a World Wide Web (WWW).

    Ele não criou a internet (que já existia com o TCP/IP), mas sim um sistema fácil de usar baseado em hipertexto.

    A WWW permitiu que documentos fossem interligados por links clicáveis, formando uma "teia" de informações.



  4. O Nascimento do HTML

    Para que a WWW funcionasse, Tim Berners-Lee precisou inventar os pilares que conectam a web até hoje:

    HTML (HyperText Markup Language): A linguagem de marcação que permite criar a estrutura dos documentos e formatar textos e links na tela.

    HTTP (HyperText Transfer Protocol): O protocolo da camada de aplicação que permite que o navegador (cliente) peça a página e o servidor a entregue.

    URI/URL: O endereço único de cada página na web.

    O TCP/IP atua como a infraestrutura invisível (o "transporte"), enquanto o HTML/WWW atua como a interface amigável (a "página" que você enxerga no navegador)



O que são Clients

(Voltar)

Cliente Web (Tecnologia e Programação)

No modelo cliente-servidor, é o programa ou dispositivo que "pede" a informação.

Navegadores: Exemplos clássicos incluem Google Chrome, Mozilla Firefox, e Microsoft Edge. Eles pedem as páginas web e as traduzem para você ler e interagir.

Aplicações e APIs: Qualquer aplicativo de celular ou programa que se conecta à internet para buscar dados é considerado um cliente web.

O que são Servers

(Voltar)

Um servidor na web é um sistema computacional (tanto o hardware quanto o software) que armazena arquivos de sites (como HTML, CSS, imagens) e os entrega aos dispositivos dos usuários pela internet, sempre que você digita um endereço ou clica em um link.

Para compreender o conceito de forma simples, os servidores desempenham dois papéis essenciais:

O Hardware: É um computador físico muito potente, mantido em um data center, que fica ligado 24 horas por dia e conectado à internet de alta velocidade.

O Software: É um programa (como o Apache HTTP Server ou o NGINX) que processa as solicitações que chegam dos navegadores web (os chamados clientes), como o Google Chrome ou o Microsoft Edge.

Como funciona uma requisição na prática:

O processo ocorre em milissegundos e segue uma lógica de "pedido e resposta":

Solicitação: Quando você digita o endereço de um site no seu navegador e aperta Enter, ele envia um pedido (requisição) pela internet para encontrar o servidor que hospeda aquele site.

Processamento: O software servidor recebe essa solicitação, localiza os arquivos corretos do site ou processa as informações necessárias.

Resposta: O servidor devolve esses dados para o seu navegador, que traduz os códigos e exibe a página na sua tela.

Linguagem de Programação

(Voltar)

O que é linguagem de programação e porque html não é linguagem de programação.

Uma linguagem de programação é um conjunto de regras e instruções usadas para criar algoritmos e programas. Ela possui lógica própria com tomadas de decisão, variáveis e cálculos. Já o HTML não é uma linguagem de programação, mas sim uma linguagem de marcação.

Para compreender as diferenças na prática:

O que é uma Linguagem de Programação?

É um meio de comunicação usado para dar ordens a um computador. Com ela, você cria sistemas, aplicativos e lógicas de negócios.

Características:

Possui estruturas condicionais (como se/então ou if/else), estruturas de repetição (loops) e manipulação de dados.

Exemplos: JavaScript, Python e Java.

Por que o HTML NÃO é uma linguagem de programação?

O HTML (que significa HyperText Markup Language, ou Linguagem de Marcação de Hipertexto) não é uma linguagem de programação porque não possui lógica. Você não consegue programar um cálculo matemático ou uma decisão lógica usando apenas HTML. Ele apenas organiza e dá significado (semântica) ao conteúdo de uma página.

O que é o HTML, afinal?

É uma linguagem de marcação de texto. Pense nele como o "esqueleto" ou a estrutura básica de um site. Ele usa "tags" (marcações) para instruir o navegador a exibir diferentes elementos.

Exemplo de uso: Em vez de processar uma regra de negócio, ele apenas diz ao navegador: "coloque este texto como um parágrafo aqui" ou "exiba esta imagem".

A tríade da internet funciona em conjunto: o HTML estrutura a página, o CSS cuida do visual (estilo) e o JavaScript é a verdadeira linguagem de programação que adiciona a interatividade e a inteligência.

Ferramentas Utilizadas

(Voltar)

Durante o curso, duas ferramentas se destacaram como essenciais para o aprendizado e desenvolvimento web: o Console F12 do navegador e o Visual Studio Code (VS Code).

O Console F12 e o VS Code são ferramentas complementares. O VS Code é a "oficina" onde você escreve e organiza o código do seu site, enquanto o Console F12 é o "laboratório" dentro do navegador onde você testa, depura e inspeciona esse site em tempo real.

Que é o Console (F12)?

O Console faz parte das Ferramentas de Desenvolvedor (também chamadas de DevTools) presentes em navegadores como Google Chrome, Microsoft Edge ou Firefox. Ele é acionado ao pressionar a tecla F12 ou clicar com o botão direito na página e escolher "Inspecionar".

Como funciona na criação web:

Depuração (Debugging): Permite que programadores usem comandos como \(console\.log()\) para exibir mensagens, rastrear valores de variáveis e verificar se o código está funcionando corretamente.

Teste de Código na Hora: Você pode digitar códigos JavaScript diretamente no console do navegador para ver resultados imediatos, sem precisar alterar seu arquivo original.

Exibição de Erros: Se algo der errado no seu site (como um script quebrado ou falha ao carregar uma imagem), o console exibirá uma mensagem de erro em vermelho para facilitar a identificação do problema.

Por que utilizar?

Ele é indispensável para diagnosticar e consertar problemas rapidamente. Em vez de adivinhar onde o código falhou, você usa o console para ver exatamente o que está acontecendo "por trás dos panos" no navegador do usuário.

O Visual Studio Code (VS Code) é um editor de código-fonte gratuito e de código aberto desenvolvido pela Microsoft. Ele é a principal ferramenta utilizada por programadores globalmente para criar e editar arquivos de programação.

Como funciona na criação web:

Escrita de Código: É nele que você digita os códigos HTML, CSS e JavaScript que formam o seu site.

Autocompletar e Organização: Ele possui ferramentas inteligentes que ajudam a autocompletar palavras, indentar (organizar) o código automaticamente e destacar com cores diferentes cada instrução para facilitar a leitura.

Extensões: Você pode instalar complementos gratuitos para ganhar novas funções, como simular o site abrindo ao vivo no navegador conforme você digita.

Por que utilizar?

O VS Code transforma a criação de sites em um processo muito mais rápido e organizado. Ele avisa quando você esquece uma letra ou pontuação errada antes mesmo de você abrir o site, diminuindo muito o tempo gasto procurando por erros básicos.

Estrutura Básica do HTML

(Voltar)

A estrutura básica de um documento HTML consiste na declaração do tipo de documento seguida pelas tags html, head (que guarda metadados e configurações) e (onde fica o conteúdo visível).

Tudo é organizado através de tags, que geralmente vêm em pares de abertura tag e fechamento /tag.

Exemplo de Estrutura Básica:

        
            <html>
                <head>
                    <title>Meu Primeiro Site</title>
                </head>
                <body>
                    <h1>Bem-vindo ao meu site!</h1>
                    <p>Este é o meu primeiro parágrafo.</p>
                </body>
            </html>

Tags

(Voltar)

Tags HTML são os blocos de construção básicos de uma página web. Escritas entre os sinais de menor e maior que < >, elas servem como "etiquetas" que dizem ao navegador web como estruturar, organizar e exibir o conteúdo, como textos, imagens e links.

Estrutura Básica

A maioria das tags funciona em pares: uma tag de abertura que inicia o elemento e uma tag de fechamento que o encerra. A diferença entre elas é apenas uma barra / na tag final.

Exemplo pratico:

            
                <html>
                    <head>
                        <title>Meu Primeiro Site</title>
                    </head>
                    <body>
                        <h1>Bem-vindo ao meu site!</h1>
                        <p>Este é o meu primeiro parágrafo.</p>
                    </body>
                </html>

Textos

(Voltar)

Tags de TextoSão usadas para estruturar e dar ênfase ao conteúdo escrito, como parágrafos e negrito.

Elas necessitam de tag de abertura e fechamento para indicar onde a formatação começa e termina Tags HTML.

  • Parágrafo <p>: Define um bloco de texto.
  • Negrito <b> ou <strong>: Deixa o texto em destaque.
  • Itálico <i> ou <em>: Coloca o texto em formato itálico.

Listas Ordenadas e Não Ordenadas

(Voltar)

Listas são usadas para organizar informações de forma clara e estruturada. Existem dois tipos principais:

  • Listas Não Ordenadas <ul>: Usam marcadores (bolinhas) para cada item.
  • Listas Ordenadas <ol>: Numeram os itens em sequência.

Exemplo de Lista Não Ordenada:

            
                <ul>
                    <li>Item 1</li>
                    <li>Item 2</li>
                    <li>Item 3</li>
                </ul>

Exemplo de Lista Ordenada:

            
                <ol>
                    <li>Primeiro</li>
                    <li>Segundo</li>
                    <li>Terceiro</li>
                </ol>

Links

(Voltar)

Links são elementos que permitem a navegação entre diferentes páginas ou seções de um site. Eles são criados usando a tag (âncora) e podem apontar para URLs externas, internas ou até mesmo para seções específicas dentro da mesma página.

Exemplo de Link Externo:

            
                <a href="https://www.exemplo.com">Visite o Exemplo</a>

Exemplo de Link Interno:

            
                <a href="outra_pagina.html">Ir para Outra Página</a>

Exemplo de Link para Seção Específica:

            
                <a href="#secao_especifica">Ir para Seção Específica</a>
